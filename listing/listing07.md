```go
package main

import (
	"fmt"
	"math/rand"
	"time"
)

func asChan(vs ...int) <-chan int {
	c := make(chan int)

	go func() {
		for _, v := range vs {
			c <- v
			time.Sleep(time.Duration(rand.Intn(1000)) * time.Millisecond)
		}

		close(c)
	}()
	return c
}

func merge(a, b <-chan int) <-chan int {
	c := make(chan int)
	go func() {
		for {
			select {
			case v := <-a:
				c <- v
			case v := <-b:
				c <- v
			}
		}
	}()
	return c
}

func main() {

	a := asChan(1, 3, 5, 7)
	b := asChan(2, 4 ,6, 8)
	c := merge(a, b )
	for v := range c {
		fmt.Println(v)
	}
}
```

Ответ:
```
Программа будет выводить числа 1, 2, 3, 4, 5, 6, 7, 8 в произвольном порядке.

Объяснение:
В программе определена функция asChan(), которая принимает список чисел vs ...int в качестве аргументов и возвращает канал (chan int). Внутри функции создается канал c и запускается горутина (анонимная функция), которая отправляет значения v из vs в канал c с интервалом времени, заданным случайной задержкой между 0 и 1000 миллисекунд. По завершении отправки, канал c закрывается. Функция asChan() возвращает канал c.

Затем в программе определена функция merge(), которая принимает два канала a и b и возвращает канал c. Внутри функции создается канал c и запускается горутина, которая бесконечно выполняет операцию select. В операторе select происходит выборка значений из каналов a и b. Если доступен элемент из a, он отправляется в канал c, а если доступен элемент из b, он также отправляется в канал c. Таким образом, значения из двух каналов объединяются в один выходной канал c.

В функции main() создаются два канала a и b, используя функцию asChan(), с разными наборами чисел. Затем создается канал c, объединяющий значения из каналов a и b с помощью функции merge(). Далее, используя цикл for range, значения из канала c считываются и выводятся на экран.

Так как в функции merge() используется оператор select, который выбирает доступное значение из двух каналов, то порядок вывода значений может быть случайным. В данном случае, программа будет выводить числа 1, 2, 3, 4, 5, 6, 7, 8 в произвольном порядке.
```
