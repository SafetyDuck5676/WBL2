Что выведет программа? Объяснить вывод программы.

```go
package main

func main() {
	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
	}()

	for n := range ch {
		println(n)
	}
}
```

Ответ:
```
Программа выведет числа от 0 до 9 в порядке возрастания.

Объяснение:
В программе создается канал ch с помощью функции make(). Канал используется для связи и обмена данными между разными горутинами.

Затем, в горутине, созданной с помощью ключевого слова go, запускается функция-замыкание. Внутри этой функции происходит цикл с переменной i, который итерирует значения от 0 до 9. В каждой итерации значение i отправляется в канал ch с помощью оператора <- и получает значение из канала.

В основной функции также используется цикл for, но с использованием конструкции range. Он продолжается до тех пор, пока получает значения из канала ch. Когда значения больше не поступают в канал, цикл завершается.

Внутри цикла основной функции каждое полученное значение n выводится с помощью функции println().

Таким образом, программа будет выводить числа от 0 до 9, поскольку каждое значение, отправленное в канал ch, будет получено и выведено в основной функции.

```
