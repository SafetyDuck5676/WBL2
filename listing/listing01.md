Что выведет программа? Объяснить вывод программы.

```go
package main

import (
    "fmt"
)

func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```

Ответ:
```
Программа выведет следующий результат:

[77 78 79]

Объяснение:
В данной программе определен массив a содержащий пять элементов типа int: 76, 77, 78, 79, 80.
Затем, задана переменная b которая будет ссылаться на срез (slice) a[1:4].
Срез a[1:4] означает, что b будет ссылаться на элементы a начиная со второго элемента (77) и до четвертого элемента (79), исключая последний (80).
В итоге, функция fmt.Println() выводит содержимое среза b, которое состоит из трех чисел: 77, 78 и 79.

```
